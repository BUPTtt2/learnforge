# 里程碑2：集成知识图谱

## 一、目标

实现知识图谱的真正集成，包括：
1. 学习开始时查询知识点掌握度
2. 根据掌握度自动调整学习模式
3. 学习完成后更新掌握度
4. 提供掌握度查询接口

## 二、修改内容

### 1. 修改 `agents/commander.py`

**学习开始时查询知识图谱**：
```python
try:
    from memory.knowledge_graph import knowledge_graph
    node = knowledge_graph.get_node(knowledge_point)
    
    if node:
        mastery_level = node.get('properties', {}).get('mastery_level', 0.0)
        # 根据掌握度调整学习模式
        if mastery_level > 0.7:
            learning_mode = "deep"
        elif mastery_level < 0.3:
            learning_mode = "simple"
    else:
        # 知识点不存在，添加到知识图谱
        knowledge_graph.add_node(knowledge_point, parent="计算机科学", properties={
            "status": "学习中",
            "mastery_level": 0.0,
            "learning_mode": learning_mode
        })
except Exception as e:
    logger.warning(f"[知识图谱] 查询失败: {e}")
```

**学习完成后更新掌握度**：
```python
def _update_knowledge_graph(self, knowledge_point: str, increment: float) -> None:
    try:
        from memory.knowledge_graph import knowledge_graph
        node = knowledge_graph.get_node(knowledge_point)
        if node:
            current_level = node.get('properties', {}).get('mastery_level', 0.0)
            new_level = min(1.0, current_level + increment)
            knowledge_graph.update_mastery_level(knowledge_point, new_level)
            
            status = "已掌握" if new_level >= 0.8 else ("学习中" if new_level > 0 else "未学")
            knowledge_graph.update_node(knowledge_point, {"status": status})
    except Exception as e:
        logger.warning(f"[知识图谱] 更新失败: {e}")
```

### 2. 修改 `ui/api.py`

**新增掌握度查询接口**：
```python
@app.route('/api/learning/mastery', methods=['GET'])
def get_mastery():
    from memory.knowledge_graph import knowledge_graph
    
    knowledge_point = request.args.get('knowledge_point')
    
    if knowledge_point:
        # 查询单个知识点
        node = knowledge_graph.get_node(knowledge_point)
        return jsonify({
            'success': True,
            'mastery_level': node.get('properties', {}).get('mastery_level', 0.0),
            'status': node.get('properties', {}).get('status', '未学')
        })
    else:
        # 返回所有知识点
        all_nodes = knowledge_graph.search_nodes('', limit=100)
        return jsonify({'success': True, 'data': all_nodes})
```

## 四、功能说明

1. **掌握度调整逻辑**：
   - 掌握度 > 0.7 → 深度模式
   - 掌握度 < 0.3 → 简单模式
   - 学习完成后掌握度 +0.3（通过）或 +0.2（部分完成）

2. **状态自动切换**：
   - 掌握度 >= 0.8 → "已掌握"
   - 掌握度 > 0 → "学习中"
   - 掌握度 = 0 → "未学"

3. **新增 API 接口**：
   - `GET /api/learning/mastery` - 获取掌握度信息
   - 参数：`knowledge_point`（可选）
