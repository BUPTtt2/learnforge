# 里程碑1：聊天历史参与 RAG 检索

## 一、目标

实现聊天历史自动存入 RAG 向量数据库，并在对话时能检索到相关历史对话作为上下文。

## 二、修改内容

### 1. 修改 `utils/chat_history.py`

在 `add_message` 方法中添加了 RAG 存储逻辑：

```python
# 新增：将对话内容存入 RAG 向量库
try:
    from utils.rag import rag_system
    metadata = {
        "type": "chat",
        "conversation_id": conversation_id,
        "role": role,
        "timestamp": datetime.now().isoformat()
    }
    rag_system.add_document(content, metadata)
except Exception as e:
    print(f"[Warning] Failed to add chat message to RAG: {e}")
```

### 2. 修改 `ui/api.py`

在 `/api/chat` 接口中添加了 RAG 检索逻辑：

```python
# 从 RAG 检索相关的对话历史
try:
    from utils.rag import rag_system
    rag_results = rag_system.search(message, top_k=2)
    if rag_results:
        related_chats = []
        for r in rag_results:
            if r.get('metadata', {}).get('type') == 'chat':
                related_chats.append(r['content'])
        if related_chats:
            context_info = "\n".join([f"相关对话: {chat}" for chat in related_chats])
```

## 四、功能说明

1. **存储时机**：每次调用 `add_message` 时自动存入 RAG
2. **检索时机**：每次调用 `/api/chat` 时自动检索相关历史
3. **检索范围**：仅检索类型为 `chat` 的文档
4. **返回数量**：最多返回 2 条相关对话
