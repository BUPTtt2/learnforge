# LearnForge简历架构图

## 系统架构图

```
┌─────────────────────────────────────────┐
│           用户交互层                    │
│  Web UI | API | 聊天接口              │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│        Agent Teams 协作层              │
│  Commander ←→ Explainer ←→ Reviewer    │
│        ↑                               │
│    Memory Manager                      │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│           RAG 检索层                    │
│  本地RAG | 外部RAG | 混合搜索          │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│           持久化层                      │
│  知识图谱 | 用户历史 | 缓存数据          │
└─────────────────────────────────────────┘
```

```

                  ↓
```

## 三大问题解决方案

```
缓存穿透 → 空值缓存
缓存击穿 → 互斥锁+双重检查
缓存雪崩 → 随机TTL波动
```
