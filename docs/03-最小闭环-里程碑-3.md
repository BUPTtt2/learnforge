# 里程碑3：分离短期/长期记忆

## 一、目标

实现短期记忆和长期记忆的分离管理：
1. 创建统一的记忆管理器
2. 短期记忆：内存存储，会话结束后可清除
3. 长期记忆：持久化存储（知识图谱）
4. 提供记忆管理 API 接口

### 2. 修改 `agents/commander.py`

使用统一记忆管理器替代直接调用知识图谱：
```python
# 学习开始时查询长期记忆
node = self.memory.get_long_term(knowledge_point)
if node:
    mastery_level = node.get('properties', {}).get('mastery_level', 0.0)

# 学习完成后更新长期记忆
self.memory.update_mastery(knowledge_point, 0.3)

# 记录到短期记忆
self.memory.add_short_term("learning_goal", learning_goal, knowledge_point)
```

### 3. 修改 `ui/api.py`

**新增记忆管理接口**：
- `GET /api/memory/short` - 获取短期记忆
- `GET /api/memory/long` - 获取长期记忆  
- `GET /api/memory/stats` - 获取记忆统计
- `POST /api/memory/clear/short` - 清除短期记忆
